# 命名空间

## 什么是命名空间

命名空间（namespace）是 Minecraft 用于区分不同来源内容的标识机制。每个数据包定义自己的命名空间，防止不同数据包之间的名称冲突。

命名空间 ID 的格式为 `<命名空间>:<路径>`，例如 `minecraft:diamond` 表示原版钻石物品，`my_datapack:explosion` 表示自定义数据包中的爆炸函数。

## 命名规则

命名空间名称只能包含小写字母、数字、下划线、连字符和点号（`[a-z0-9_.-]`）。路径部分额外支持 `/`。

## 在 DynFirework 中的使用

DynFirework 导出的数据包使用独立的命名空间（默认 `fireworks1`），所有 function 文件存放在该命名空间下，与游戏原版及其他数据包的内容隔离。

导出时的命名空间可在导出设置中自定义。
