# DF 轨迹参数

选中轨迹元素后，参数面板显示以下可调参数。不同类型的轨迹显示不同的参数组。

## 基本信息

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 元素的显示名称 | 轨迹 N |
| 开始时间 | 轨迹开始的时刻（秒） | 0.0 |
| 持续时间 | 轨迹持续的时间（秒） | 2.0 |

## 位置

| 参数 | 说明 |
|------|------|
| X、Y、Z（起始） | 轨迹起点的三维坐标 |
| X、Y、Z（结束） | 轨迹终点的三维坐标 |
| 在地图上选择 | 打开位置选择器选取坐标 |

模组使用二分法从终点坐标反算初速度，使轨迹恰好到达目标点。

## 颜色

仅 `launch` 支持自定义渐变色。其余轨迹类型（`launch_spark` 固定暖白→金黄色，`expanding` 和 `spiral` 无颜色参数）。

| 参数 | 适用于 | 说明 |
|------|--------|------|
| 起始色 R/G/B | launch | 粒子出现时的颜色（0-255） |
| 结束色 R/G/B | launch | 粒子消失时的颜色（0-255） |
| 使用渐变 | launch | 启用后颜色从起始色过渡到结束色 |

## 物理参数

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 阻力系数 k | 所有类型 | 空气阻力系数 | 1.2 |
| 质量 m0 | 所有类型 | 粒子质量 | 0.5 |
| 粒子密度 rho | launch | 沿轨迹采样密度（值越小越密） | 1.0 |
| 粒子数量 | launch_spark、expanding、spiral | 每采样点生成的粒子数 | 1 |
| 扩散 X/Y/Z | expanding | 三个方向的随机偏移范围 | 0.0 |
| 速度因子 | expanding | 膨胀速度缩放系数 | 1.0 |
| 螺旋半径 | spiral | 螺旋旋转半径 | 2.0 |
| 螺旋速度 | spiral | 螺旋旋转角速度 | 3.0 |
| 收缩指数 | spiral | 半径收缩指数（值越大收缩越早） | 1.0 |
| 生命周期 | launch、expanding、spiral | 粒子存活时间（秒） | 3.0 |
| 添加弹体 | 所有类型 | 在终点生成弹体粒子 | 否 |
| 弹体颜色 R/G/B | launch | 弹体粒子颜色 | 255 165 0 |
| 弹体大小 | launch | 弹体粒子大小 | 1.0 |

轨迹子类型：`launch`、`launch_spark`、`expanding`、`spiral`。通过 `/df` 命令生成单条指令。
