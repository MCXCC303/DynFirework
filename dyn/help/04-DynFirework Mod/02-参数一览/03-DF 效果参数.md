# DF 效果参数

选中效果元素后，参数面板显示以下可调参数。不同类型的效果显示不同的参数组。

## 基本信息

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 元素的显示名称 | 效果 N |
| 开始时间 | 效果开始的时刻（秒） | 0.0 |
| 持续时间 | 效果的持续时间（秒） | 2.0 |

## 位置

| 效果 | 位置参数 | 说明 |
|------|----------|------|
| beam | X、Y、Z | 束状发射原点 |
| spray | X、Y、Z | 喷射原点 |
| double_helix | X、Z、基准 Y | 螺旋中心轴位置（X、Z、底部 Y） |
| rotating_ring | X、Y、Z | 环中心坐标 |

## 颜色

| 参数 | 适用于 | 说明 |
|------|--------|------|
| 起始色 R/G/B | beam、spray | 粒子出现时的颜色（3 个 0-255 整数） |
| 结束色 R/G/B | beam、spray | 粒子消失时的颜色（3 个 0-255 整数） |
| 颜色组 1 起始/结束 R/G/B | double_helix | 第一组螺旋粒子的起止色（6 个值） |
| 颜色组 2 起始/结束 R/G/B | double_helix | 第二组螺旋粒子的起止色（6 个值） |
| 起始色 R/G/B | rotating_ring | 环面粒子出现时的颜色 |
| 结束色 R/G/B | rotating_ring | 环面粒子消失时的颜色 |

## 物理参数

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 最小速度 | beam、spray | 粒子最小初速度 | 1.0 |
| 最大速度 | beam、spray | 粒子最大初速度 | 3.0 |
| 水平角度 | beam、spray | 水平发射方向（度） | 0.0 |
| 垂直角度 | beam、spray | 垂直发射方向（度） | 0.0 |
| 扩散角 | beam | 束状锥角（度），80% 粒子均匀填充 | 15.0 |
| 锥角 | spray | 喷射锥角（度） | 15.0 |
| 环半径 | double_helix | 螺旋环绕半径 | 2.0 |
| 上升速度 | double_helix | 螺旋上升速度 | 3.0 |
| 旋转速度 | double_helix、rotating_ring | 螺旋/环面旋转角速度 | 5.0 / 2.0 |
| 环面半径 | rotating_ring | 环面主半径 | 3.0 |
| 管径 | rotating_ring | 环面管径 | 0.5 |
| 径向速度 | rotating_ring | 粒子径向扩散速度 | 1.0 |

## 数量

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 束数 | beam | 发射束数量 | 8 |
| 每束粒子数 | beam | 每束包含粒子数 | 10 |
| 每 tick 粒子数 | spray | 每 tick 喷射粒子数 | 3 |
| 密度 | double_helix、rotating_ring | 粒子密度 | 20 / 30 |

## 时间参数（tick 单位）

注意：以下参数单位为 **tick**（20 tick = 1 秒），非秒。

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 喷射时长 | spray | 持续喷射的 tick 数 | 40 |
| 粒子生命周期 | spray | 每个粒子的存活 tick 数 | 20 |
| 持续 tick 数 | double_helix、rotating_ring | 效果持续的 tick 数 | 100 |

效果子类型包括 `beam`、`spray`、`double_helix`、`rotating_ring`，通过 `/df` 命令生成单条指令。
