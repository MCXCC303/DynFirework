# DF 复合参数

选中复合元素后，参数面板显示以下可调参数。不同类型显示不同的参数组。每个复合元素对应一条 `/df` 命令，内部协调由模组处理。

## 基本信息

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 元素的显示名称 | 复合 N |
| 开始时间 | 复合效果开始的时刻（秒） | 0.0 |
| 持续时间 | 复合效果的持续时间（秒） | 2.0 |

## 二次爆炸（secondary_explosion）

主烟花在指定位置爆炸后，在每个粒子终点触发二次爆炸。主烟花固定为 `single_layer`，二次爆炸可选 `expanding` 或 `single_layer`。所有时间单位为秒。

### 主烟花参数（single_layer）

| 参数 | 说明 | 默认 |
|------|------|------|
| 爆炸位置 X/Y/Z | 主烟花爆炸中心 | 0, 64, 0 |
| 起始色 R/G/B | 主烟花粒子出现时的颜色 | 255 165 0 |
| 结束色 R/G/B | 主烟花粒子消失时的颜色 | 255 0 0 |
| 速度 | 主烟花粒子初速度 | 10.0 |
| 粒子总数 | 主烟花粒子总数 | 100 |
| 持续时间 | 主烟花效果持续秒数 | 2.0 |
| 生命周期 | 主烟花粒子存活秒数 | 2.0 |
| 尾部闪烁 | 粒子尾部闪烁效果 | 否 |

### 二次爆炸参数

**类型 1 — expanding（扩散球面）：**

| 参数 | 说明 | 默认 |
|------|------|------|
| 半径 | 球面初始半径 | 3.0 |
| 粒子数 | 球面粒子总数 | 50 |
| 径向速度 | 向外膨胀速度 | 2.0 |
| 生命周期 | 粒子存活秒数 | 1.5 |

**类型 2 — single_layer（单层烟花）：**

| 参数 | 说明 | 默认 |
|------|------|------|
| 起始色 R/G/B | 二次烟花粒子出现时的颜色 | 255 255 0 |
| 结束色 R/G/B | 二次烟花粒子消失时的颜色 | 255 100 0 |
| 速度 | 粒子初速度 | 8.0 |
| 粒子数 | 粒子总数 | 50 |
| 持续时间 | 效果持续秒数 | 1.5 |
| 生命周期 | 粒子存活秒数 | 1.5 |

## 同步烟花（combo_ec）

集束烟花与扩散球面在相同位置同步展开。球面速度由模组自动计算。

| 参数 | 说明 | 默认 |
|------|------|------|
| 中心位置 X/Y/Z | 爆炸中心 | 0, 64, 0 |
| 集束起始色 R/G/B | 集束粒子出现时的颜色 | 255 0 0 |
| 集束结束色 R/G/B | 集束粒子消失时的颜色 | 255 100 0 |
| 集束速度 | 集束粒子初速度 | 10.0 |
| 方向数 | 集束方向数量 | 12 |
| 轨迹数 | 每方向轨迹数 | 5 |
| 扩散角 | 集束发散度（度） | 15.0 |
| 持续时间 | 效果持续秒数 | 2.0 |
| 生命周期 | 所有粒子存活秒数 | 2.0 |
| 球面起始色 R/G/B | 球面粒子出现时的颜色 | 0 0 255 |
| 球面结束色 R/G/B | 球面粒子消失时的颜色 | 100 100 255 |
| 球面粒子数 | 球面粒子总数 | 100 |
| 尾部闪烁 | 球面粒子尾部闪烁 | 否 |
