# DF 烟花参数

选中烟花元素后，参数面板显示以下可调参数。不同类型的烟花显示不同的参数组。

## 基本信息

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 元素的显示名称 | 烟花 N |
| 开始时间 | 烟花爆炸发生的时刻（秒） | 0.0 |
| 持续时间 | 烟花效果的持续时间（秒） | 2.0 |

## 爆炸中心

| 参数 | 说明 |
|------|------|
| X、Y、Z | 爆炸中心的三维坐标 |
| 在地图上选择 | 打开位置选择器选取爆炸位置 |

## 颜色

所有烟花类型使用一组渐变色（起始色 → 结束色，各 R/G/B 0-255）。双层烟花额外包含外层独立渐变色。

| 参数 | 适用于 | 说明 |
|------|--------|------|
| 起始色 R/G/B | 所有类型 | 粒子出现时的颜色 |
| 结束色 R/G/B | 所有类型 | 粒子消失时的颜色 |
| 外层起始色 R/G/B | double_layer | 外层球面粒子出现时的颜色 |
| 外层结束色 R/G/B | double_layer | 外层球面粒子消失时的颜色 |
| 使用渐变 | 所有类型 | 启用后颜色从起始色过渡到结束色 |

## 物理参数

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 速度 | single_layer、directional、clustered | 粒子初速度 | 10.0 |
| 内层速度 | double_layer | 内层球面粒子初速度 | 8.0 |
| 外层速度 | double_layer | 外层球面粒子初速度 | 10.0 |
| 粒子总数 | single_layer、double_layer（各层独立）、expanding_sphere、nebula | 粒子总数 | 100 |
| 方向数 | clustered | 斐波那契球面分布方向数 | 12 |
| 轨迹数 | directional、clustered | 每方向粒子轨迹数 | 5 |
| 方向水平角 | directional | 发射方向水平角（度） | 0.0 |
| 方向垂直角 | directional | 发射方向垂直角（度） | 0.0 |
| 扩散角 | directional、clustered | 方向扩散范围（度） | 15.0 |
| 半径 | expanding_sphere | 初始球面半径 | 5.0 |
| 径向速度 | expanding_sphere | 粒子向外膨胀的径向速度 | 3.0 |
| 膨胀速度 | nebula | 星云膨胀速度 | 2.0 |
| 密度衰减指数 | nebula | 幂律密度衰减（值越大中心越密集） | 2.0 |
| 持续时间 | single_layer、double_layer、directional、clustered | 效果持续秒数 | 2.0 |
| 生命周期 | 所有类型 | 粒子存活时间（秒） | 2.0 |
| 尾部闪烁 | single_layer、double_layer、directional、clustered、expanding_sphere | 粒子尾部闪烁效果 | 否 |

烟花子类型：`single_layer`、`double_layer`、`directional`、`clustered`、`expanding_sphere`、`nebula`。通过 `/df` 命令生成单条指令。
