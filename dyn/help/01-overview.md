# 概述

DynFirework 是一款为 Minecraft 设计的烟花粒子秀编辑工具。

## 功能

在 Minecraft 中实现粒子烟花秀，传统方式需要手写数百条命令。DynFirework 将这些计算工作自动化，在时间线上拖拽编排烟花效果，最后导出为 Minecraft 数据包。

## 工作流

1. **创建项目**：设置项目名称、BPM 和目标 Minecraft 版本。
2. **导入音频**（可选）：加载音乐文件用于节拍对齐。
3. **创建元素**：添加烟花和轨迹元素。
4. **调整参数**：修改元素的各项属性。
5. **导出数据包**：生成适配目标版本的 Minecraft 数据包。

## 界面布局

主窗口分为五个区域：

- **元素列表**：左侧，树形结构展示所有元素。
- **参数面板**：中央上方，编辑选中元素的属性。
- **检查器**：右侧，以 JSON 展示选中元素的数据。
- **播放控制栏**：提供音频播放和音量控制。
- **时间线**：底部，以水平色块显示元素的时间范围。烟花在上半区域，轨迹在下半区域。

## 项目文件

DynFirework 使用 `.dyn` 后缀的压缩归档文件（tar.gz 格式）保存项目。文件内部包含：
- `manifest.json` - 项目元数据、文件清单和 SHA-256 校验和
- `elements/` - 每个元素独立存储为 JSON 文件
- `positions.json` - 保存的位置坐标
- `timeline_state.json` - 时间线 UI 状态
- `assets/` - 嵌入的音乐等资源文件

音乐文件嵌入在归档中，移动项目文件不会丢失音乐。

## 支持的 Minecraft 版本

DynFirework 支持两种 Minecraft 粒子模组，在项目设置中选择目标版本后自动适配：

| Minecraft 版本 | 粒子模组 | 命令格式 |
|---------------|---------|---------|
| 1.12.2 / 1.16.5 | Colorblock Mod | `/particleex` |
| 1.20.1 / 1.20.4 / 1.21 | DynFirework Particles Mod | `/dfp` |

详见第 2.3 章：Minecraft 命令。
