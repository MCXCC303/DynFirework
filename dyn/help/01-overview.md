# 概述

DynFirework 是一款为 Minecraft 设计的烟花粒子秀编辑工具。

## 功能

在 Minecraft 中实现粒子烟花秀，传统方式需要手写数百条命令。DynFirework 将这些计算工作自动化，在时间线上拖拽编排烟花效果，最后导出为 Minecraft 数据包。

## 工作流

1. **导入音频**（可选）：加载音乐文件用于节拍对齐。
2. **创建元素**：添加烟花和轨迹元素。
3. **调整参数**：修改元素的各项属性。
4. **导出数据包**：生成 Minecraft 可加载的数据包。

## 界面布局

主窗口分为五个区域：

- **元素列表**：左侧，树形结构展示所有元素。
- **参数面板**：中央上方，编辑选中元素的属性。
- **检查器**：右侧，以 JSON 展示选中元素的数据。
- **播放控制栏**：提供音频播放和音量控制。
- **时间线**：底部，以水平色块显示元素的时间范围。烟花在上半区域，轨迹在下半区域。

## 项目文件

DynFirework 使用 `.dyn` 后缀的 JSON 文件保存项目。文件包含所有元素的配置数据和音乐文件路径引用。音乐不嵌入项目文件。
