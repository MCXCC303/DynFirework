# 轨迹参数一览

以下参数适用于 ColorBlock 后端，时间单位均为 tick。

选中轨迹元素后，参数面板显示以下可调参数。不同类型的轨迹显示不同的参数组。

## 基本信息

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 名称 | 元素的显示名称 | 轨迹 N |
| 起始 Tick | 轨迹开始出现的时刻 | 0 |
| 时长 | 轨迹持续的 tick 数 | 20 |

## 位置

| 参数 | 说明 |
|------|------|
| X、Y、Z（起始） | 轨迹起点的三维坐标 |
| X、Y、Z（结束） | 轨迹终点的三维坐标 |
| 在地图上选择 | 打开位置选择器选取坐标 |

## 颜色

| 参数 | 说明 |
|------|------|
| 使用渐变 | 启用后粒子颜色在生命周期内从起始色过渡到结束色 |
| 起始颜色 | 粒子出现时的颜色（R/G/B，0 至 255） |
| 结束颜色 | 渐变启用时，粒子消失前的颜色 |

## 物理参数

| 参数 | 适用于 | 说明 | 默认 |
|------|--------|------|------|
| 阻力系数 k | 所有类型 | 空气阻力系数 | 1.2 |
| 质量 m0 | 所有类型 | 粒子质量 | 0.5 |
| 粒子密度 rho | launch | 每 tick 生成的粒子数 | 1 |
| 粒子数量 | spark、thick、expanding | 每采样点生成的粒子数 | 1 |
| 偏移间隔 | offset、thick、expanding | 随机偏移采样间隔（tick） | 5 |
| 每 tick 点数 | offset、thick、expanding | 每 tick 样条插值点数 | 1 |
| 扩散 X/Y/Z | thick、expanding | 三个方向的随机扩散范围 | 0 |
| 速度因子 | expanding | 膨胀速度缩放系数 | 1.0 |
