# 位置选择器

位置选择器是一个独立工具，用于在二维平面上图形化选取三维坐标。从元素参数中的「在地图上选择」按钮打开。

## 视图模式

在菜单栏的「视图」菜单中切换模式。

### 网格图模式（默认）

笛卡尔坐标网格。X 轴向右，Z 轴向下，主刻度线每 16 格一条，次刻度线每 1 格一条。

### 雷达图模式

极坐标视图：同心圆环（间距 16 格）和放射线（间距 30 度）。

## 操作

| 操作 | 方式 |
|------|------|
| 选中已有的点 | 左键点击 |
| 创建新点 | 左键点击空白位置 |
| 编辑点 | 右键，选择「编辑点」 |
| 删除点 | 右键，选择「删除点」 |
| 复制坐标 | 右键，选择「复制坐标」 |
| 平移视图 | 按住鼠标中键拖拽 |
| 缩放视图 | 鼠标滚轮 |
| 重置视图 | 右键，选择「重置视图」 |

## 创建

点击空白位置开始创建，输入名称、X/Y/Z 三维坐标，以及颜色设置。

右侧列表显示所有已保存的点，确认选择后坐标传回主窗口。
