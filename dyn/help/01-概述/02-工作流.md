# 工作流

从新建项目到导出数据包的完整流程分为以下步骤：

1. **创建项目**：设置项目名称、BPM，选择模组后端（ColorBlock 或 DynFireworkMod）和目标 Minecraft 版本。后端在创建后锁定，不可更改。
2. **导入音频**（可选）：加载 MP3、WAV、OGG 或 FLAC 格式的音乐文件，用于在时间线上对齐节拍。
3. **创建元素**：通过元素浏览器添加烟花、轨迹、效果或复合元素。每种元素类型有各自的子类型，对应不同的粒子形态。
4. **编辑参数**：在属性面板中调整选中元素的各项参数，包括位置、颜色、速度、角度等。使用位置选择器在地图上直观拾取坐标。
5. **编排时间线**：在时间线上拖拽色块调整元素的起始时刻和持续时长，缩放和滚动视图查看全局排列。
6. **导出数据包**：导出为适配目标 Minecraft 版本的数据包。生成的 `.mcfunction` 文件按 tick 组织，使用 `schedule` 命令串联播放。
