# 界面布局

主窗口分为五个主要区域：

- **元素浏览器**（左侧）：树形结构展示项目中所有元素，按烟花、轨迹等分类排列。选中 ColorBlock 的复合烟花元素或 DF 的复合元素时，展开显示子节点，用于分别编辑子元素的参数。
- **属性面板**（中央）：上下文敏感的表单区域。根据选中的元素类型和子类型动态切换对应的编辑表单，实时修改元素属性。
- **检查器**（右侧）：以只读 JSON 格式展示选中元素的完整数据，方便调试和查看全部字段。
- **播放控制栏**（顶部）：音频播放控制按钮（播放、暂停、停止）、BPM 显示、音量滑块和当前时间指示。
- **时间线**（底部）：水平轨道以彩色色块表示每个元素的时间范围。ColorBlock 后端有烟花和轨迹两条轨道；DynFireworkMod 后端有烟花、轨迹、效果和复合共四条轨道。时间线上还显示节拍线和播放头位置。
