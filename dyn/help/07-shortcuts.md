# 快捷键

## 文件操作

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+N` | 新建项目 |
| `Ctrl+O` | 打开项目 |
| `Ctrl+S` | 保存项目 |
| `Ctrl+Shift+S` | 另存为 |
| `Ctrl+E` | 导出数据包 |

## 编辑操作

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+Shift+T` | 新建轨迹 |
| `Ctrl+Shift+F` | 新建烟花 |
| `Ctrl+C` | 复制选中元素 |
| `Delete` | 删除选中元素 |

## 播放控制

| 快捷键 | 操作 |
|--------|------|
| `Space` | 播放或暂停 |

## 位置选择器

位置选择器窗口获得焦点时：

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
